The kinematic and dynamic analysis of four bar mechanism are presented. The Forward Kinematic analysis is done using Raven's Loop Clousure Equations, Newton-Raphson, Denavit-Hartenberg (D-H) representation and Geometric Jacobian methods, Inverse Kinematic Analysis is done using Geometric Jacobian. The Inverse Kinetic (Dynamic) Analysis is done with deriving Newton-Euler Equilibrium equations and results are validated with Lagrangian formulation method whereas the Forward Dynamic Analysis with Lagrangian method.
